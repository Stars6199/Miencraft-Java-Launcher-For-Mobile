# Minecraft-Java-Edition-Launchers-Android

A list of 3rd party launchers for Minecraft Java Edition on Android.


| Name | Description & Features | Source |
| :--- | :--- | :--- |
| **Pojav Launcher** | A Minecraft: Java Edition Launcher for Android and iOS based on Boardwalk. It supports almost all versions from early builds to the latest snapshots, including Forge and Fabric support. **Note:** This project is discontinued. | [GitHub](https://github.com/PojavLauncherTeam/PojavLauncher) / [Discord](https://discord.com/invite/aenk3EUvER) |
| **Amethyst Launcher** | A Minecraft: Java Edition Launcher for Android and iOS based on PojavLauncher. It is optimized for jailbroken and TrollStore devices and supports Forge, Fabric, OptiFine, and Quilt for extensive modding. | [GitHub](https://github.com/AngelAuraMC/Amethyst-Android) / [Discord](https://discord.gg/5ptqkyZxEy) |
| **Mojo Launcher** | A lightweight Minecraft: Java Edition launcher based on PojavLauncher. It provides a focused environment for running LWJGL-based games with community-driven updates. | [GitHub](https://github.com/MojoLauncher/MojoLauncher) / [Discord](https://discord.gg/VHdwQFsaGX) |
| **Zalith Launcher 1** | A community remake based on PojavLauncher that aims to enhance user experience. It features a redesigned app interface and practical tools to make running Java Edition easier for mobile users. | [GitHub](https://github.com/ZalithLauncher/ZalithLauncher) / [Discord](https://discord.gg/e7C4kytRgK) / [Website](https://www.zalithlauncher.cn/en/download.html) |
| **Zalith Launcher 2** | A next-generation launcher for Android built with modern technologies like Jetpack Compose and Material Design 3. It offers improved performance and a highly polished UI. | [GitHub](https://github.com/ZalithLauncher/ZalithLauncher2) / [Discord](https://discord.gg/e7C4kytRgK) / [Website](https://www.zalithlauncher.cn/en/) |
| **MCinaBox** | An encapsulation of CosineMath's BoatApp project. It provides full user management, Minecraft version management, virtual game controllers, and JRE runtime environment configurations. **Note:** This project is discontinued. | [GitHub](https://github.com/AOF-Dev/MCinaBox) / [Discord](https://discord.gg/55GECvHuNd) |
