# Minecraft-Java-Edition-Launchers-Android

A list of 3rd party launchers for Minecraft Java Edition on Android and iOS devices.

## Android & iOS Launchers

| Name | Description | Features | Source |
| :--- | :--- | :--- | :--- |
| **PojavLauncher** | A Minecraft: Java Edition Launcher for Android and iOS based on Boardwalk. | Succeeded by Amethyst; runs almost all versions of Java Edition. | [GitHub](https://github.com/PojavLauncherTeam/PojavLauncher) |
| **Amethyst Launcher** | A Minecraft: Java Edition Launcher for Android and iOS based on PojavLauncher. | Successor to Pojav; focused on modern Java support and stability. | [GitHub](https://github.com/AngelAuraMC/Amethyst-Android) |
| **MojoLauncher** | A Minecraft: Java Edition launcher, based on PojavLauncher. | Lightweight mobile launcher for LWJGL-based games. | [GitHub](https://github.com/MojoLauncher/MojoLauncher) |
| **ZalithLauncher 1** | A Minecraft launcher developed based on PojavLauncher. | Redesigned interface and enhanced user experience for Android. | [GitHub](https://github.com/ZalithLauncher/ZalithLauncher) |
| **ZalithLauncher 2** | A newly designed Minecraft: Java Edition Launcher for Android. | Built with Jetpack Compose and Material Design 3. | [GitHub](https://github.com/ZalithLauncher/ZalithLauncher2) |
| **MCinaBox** | A Minecraft: Java Edition Launcher for Android. | Encapsulation of BoatApp; focuses on running the original Java game. | [GitHub](https://github.com/AOF-Dev/MCinaBox) |
